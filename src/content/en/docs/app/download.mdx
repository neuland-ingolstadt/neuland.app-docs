## iOS

The app is available for iOS and iPadOS. You can download the app from the Apple App Store.

The minimum required iOS version is 15.1.


## Android

The app is available for Android phones and tablets. You can download it either from the Play Store or directly from source. 

The minimum required Android version is 7.0.

### Play Store

You can download the latest version of the app directly from the Google Play Store using the button above.

Alternatively, you can use [Aurora Store](https://auroraoss.com/), an open-source alternative to the Google Play Store that allows you to download apps without a Google account. Just search for `Neuland Next` in the Aurora Store and install it.

### GitHub Releases

The app is also available as a universal `.apk` file on the [GitHub Releases page](https://github.com/neuland-ingolstadt/neuland.app-native/releases). This option offers maximum flexibility and transparency, allowing you to install the app on any Android device without relying on the Play Store.

To automate update installation, you can use a third-party app like [Obtainium](https://github.com/ImranR98/Obtainium), which automatically fetches updates from GitHub Releases. Simply add the repository link:  
`https://github.com/neuland-ingolstadt/neuland.app-native`.

## MacOS

The app is also available for MacOS running on Apple Silicon. You can download the app from the Mac App Store.

Neuland Next is primarily designed for mobile devices. The MacOS version is the iPadOS version running on MacOS. It is not fully optimized for desktop use.


## Windows & Linux

The app is not available for Windows and Linux.

We recommend using the web version of the app on your device. See [Web Version](#web-version).

## Web Version

You can use the web version of Neuland Next on any device with a modern web browser. It can be accessed at [neuland.app](https://web.neuland.app).

Note that the web version does not support all features of the native app. For the best experience, we recommend using the native app on your device.
